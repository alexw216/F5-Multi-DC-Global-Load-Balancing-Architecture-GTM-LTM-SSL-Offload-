# Contributing

Thank you for your interest in improving this F5 Multi-DC GSLB reference.

## How to contribute

- Open an issue to propose changes or new scenarios.
- Fork the repo and create a feature branch.
- Keep changes small and focused (docs, configs, or diagrams).
- Update relevant docs and diagrams when changing configs.

## Coding / config style

- Use tmsh syntax consistently.
- Prefer parameterized examples (clearly mark placeholders).
- Keep diagrams in both ASCII and Mermaid where possible.

## Pull requests

- Reference the issue you’re addressing.
- Describe the change and its impact.
- Include validation notes (what you tested).
